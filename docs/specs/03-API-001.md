# API-001 認証API

## エンドポイント

### POST /api/auth/login
- **概要**: ユーザーログイン
- **リクエスト**: `{ email: string, password: string }`
- **レスポンス**: `{ token: string, user: User }`
- **エラー**: 401 (認証失敗), 429 (レート制限)

### POST /api/auth/logout
- **概要**: ユーザーログアウト
- **リクエスト**: Authorization ヘッダー
- **レスポンス**: `{ success: true }`
